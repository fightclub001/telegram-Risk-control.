🎛️ 可配置参数总数：40+ 个
════════════════════════════════════════════════════════════════════════════

简介检测（3个）
   • check_bio_link           - 检测链接
   • check_bio_keywords       - 检测敏感词
   • bio_keywords             - 敏感词列表

显示名称检测（2个）
   • check_display_keywords   - 检测敏感词
   • display_keywords         - 敏感词列表

短消息检测（5个）
   • short_msg_detection      - 启用/禁用
   • short_msg_threshold      - 字数阈值
   • min_consecutive_count    - 连续条数
   • time_window_seconds      - 时间窗口
   • user_short_msg_history   - 历史记录

填充垃圾检测（4个）
   • fill_garbage_detection   - 启用/禁用
   • fill_garbage_min_raw_len - 最小长度
   • fill_garbage_max_clean_len - 清理后长度
   • fill_space_ratio         - 空格比例

自动回复（7个）
   • autoreply.enabled        - 启用/禁用
   • autoreply.keywords       - 关键词列表
   • autoreply.reply_text     - 回复文本
   • autoreply.buttons        - 按钮列表
   • autoreply.delete_user_sec - 用户消息删除延时
   • autoreply.delete_bot_sec  - 机器人消息删除延时

群组设置（3个）
   • enabled                  - 启用/禁用群组
   • name                     - 群组名称
   • exempt_users             - 豁免用户列表

举报系统（不需配置，自动）
   • 用户举报
   • 管理员封禁
   • 自动记录

   → 在私聊中发送（不是群组）
   → 查看 Railway 日志中的错误

   → 确保群组 ID 在 GROUP_IDS 中
   → 确保机器人是群组管理员
   → 查看群组配置是否启用

   → 检查 Railway 日志
